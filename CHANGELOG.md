# Changelog

All notable changes to the React Native Profiler AI extension will be documented in this file.

## [0.1.0] - Initial Release

### Added
- VS Code extension for React Native performance profiling
- Interactive component treeview for selecting components to profile
- Local HTTP server for receiving profiling data from React Native apps
- Real-time log display
- AI-powered performance analysis using OpenAI or Anthropic
- Automatic bottleneck detection
- Source code context inclusion in AI analysis
- Configuration management for React Native apps
- `withProfiler` HOC for easy component integration
- Support for both OpenAI and Anthropic APIs
- Auto-analysis after recording stops (configurable)
- Comprehensive documentation and examples

### Features
- Component selection via interactive tree
- Start/Stop recording controls
- Live profiling data display
- AI analysis with actionable recommendations
- Secure API key storage in VS Code settings
- Configurable server port
- Error handling and user notifications
